Silicon Valley.

BootStrap practice.


Task 1:  Getting the BootStrap starting code.

 Go to http://azulait.com/y.html, right click, view the source code and copy it to the clipboard. Paste the code to Sublime. Upload the html file to your GitHub account so it will accessible on the Internet.



Task 2:  Inserting random text.

 Go to http://randomtextgenerator.com/, generate some text in English and paste it on your web page. Generate some text in French and German and paste it too. Upload the page to GitHub.



Task 3:  Testing your web page on different mobile devices.

 Go to http://mobiletest.me, choose the IPhone 5 phone and test your page! Do the same with the HTC One, the Google Nexus 7 and the Apple IPad Mini.
