Working with the W3CSS framework.



Task 1:  Working with the W3CSS band template.

 Go to http://www.w3schools.com/css/css_rwd_templates.asp and click on the "Try it yourself" button of the Band template. Copy the whole code to a new file named band.html (created with Sublime). Create a new folder on your desktop named Band and copy the band.html file to it. Click on every picture you see on the "band" web page and download it to the Band folder. Once you've downloaded all the pictures, double click on the band.html web page. Now upload all the Band folder files to your GitHub account. Test the band.html page.



Task 2:  Working with the W3CSS house design template.

 Go to http://www.w3schools.com/css/css_rwd_templates.asp and click on the "Try it yourself" button of the house design template. Copy the whole code to a new file named housedesign.html (created with Sublime). Create a new folder on your desktop named HouseDesign and copy the housedesign.html file to it. Click on every picture you see on the "House Design" web page and download it to the HouseDesign folder. Once you've downloaded all the pictures, double click on the housedesign.html web page. Now upload all the HouseDesign folder files to your GitHub account. Test the housedesign.html page.



Task 3:  Working with the W3CSS blog, photo, start page, portfolio, parallax, marketing, kitchen sink, web page, social media, analytics, travel agency W3CSS templates.

 Go to http://www.w3schools.com/css/css_rwd_templates.asp. Follow the steps from the previous tasks to create web sites using the W3CSS templates. As usual first create a folder, name it, add pictures, and finally upload the files to your GitHub account. Test the uploaded web pages. Use all the templates (in any order you want).
