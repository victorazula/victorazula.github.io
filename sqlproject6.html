Exercise:   Exporting query results from SQL to a CSV file and then to an HTML file.

Step 1:     Create a new database named Database002.

Step 2:     Go to http://azulait.com/customers08102016.txt and copy the whole code shown there. In SSMS create a new query, paste the copied code and execute it. Verify that you have a new table named customers.

Step 3:     Create a query to find out all the customers whose first name is Catherine.

Step 4:     Right click on the results pane, select all, save results as Catherine.csv (on your documents or anywhere you want).

Step 5:     Go to http://www.convertcsv.com/csv-to-html.htm and click on "Choose File" to upload Catherine.csv. Click on the "CSV to HTML table" button and then on the "Download Result" button. Open the file that was created (an html file that shows a table with all the customers whose name is Catherine).
