Using DataTables (a JQuery plugin) to display data.

Step 1:     Create a new database named Biz.

Step 2:     Go to http://azulait.com/ourcustomers.txt and copy the whole text (by pressing CTRL A and then CTRL C). Create a new query, paste the copied code and execute it. You should have now a newly created table named OurCustomers. Verify that.

Step 3:     Find out all customers. Show all fields. Save the results as OurCustomers.csv. Transform the CSV file into an HTML file by going to http://www.convertcsv.com/csv-to-html.htm and open the web file created. Right click on the web page to view the source and see the generated table code. Keep it open.

Step 4:     We'll install a sophisticated text editor named Sublime. Get it and install it from here.

Step 5:     Go to http://azulait.com/datatablestemplate.html, right click and view the source code. Copy it.

Step 6:     Using Sublime create a new file named BizCustomers.html. Paste the code from the previous step. Copy the table code from Step 3 (that you kept open) and paste it on the proper place. Save the file. Make modifications such as creating the thead tag and adding an id to the table tag. Save the file.

Step 7:     Open the file in different browsers and sort the data, search for specific countries, define how many records are shown per page, etc.
