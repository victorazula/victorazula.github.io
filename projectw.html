Task 1:  Getting data from Mockaroo.

 Go to http://www.mockaroo.com. Use these fields: id, first_name, last_name, email and city. No need for gender nor ip_address. Generate 1000 records on a CSV file. Download the file.



Task 2:  Converting CSV to HTML.

 Go to http://www.convertcsv.com/csv-to-html.htm. Use the previously created CSV file as input. Copy the HTML code (which creates a table) and paste it on Sublime. Upload the file to GitHub.
